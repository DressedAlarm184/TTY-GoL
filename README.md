This is Conway's Game of Life but it runs in the terminal! To build, run `gcc -o main main.c` and to run it, use `./main`. When you start it, there will be an empty 35 by 20 grid. Use `WASD` to move the cursor, `Space` to set or unset a cell, and `Enter` to begin the simulation. There is a generation counter at the top and one generation passes every 150 milliseconds.
