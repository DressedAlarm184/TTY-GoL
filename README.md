This is Conway's Game of Life but it runs in the terminal! To build the program, run `gcc -o main main.c` and to start it, use `./main`. When you start it, there will be an empty 35 by 20 grid. Use `WASD` to move the cursor, `Space` to set or unset a cell, and `Enter` to begin the simulation. One generation completes every 150 milliseconds. This program depends on nothing but the standard C library. To exit, press `^C`. You cannot pause the simulation. The edges are hard walls and cells do not wrap around to the other side. Terminal must support ANSI escape codes and Unicode.
